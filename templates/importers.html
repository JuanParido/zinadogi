How to implement file uploader

To work, the file uploader uses jquery.fileuploader.min.js and jquer.fileuploader.css files which are already included in DoGi. You can take a look to the documentation if needed at https://innostudio.de/fileuploader/documentation/.

Inside the template file the uploader must be constructed as follows:


                    <form action="#" method="post" enctype="multipart/form-data">
                      <label for="files">File uploader</label>
                      <input type="file" name="files">
                      <button class="btn btn-success pull-right import-send-btn" type="submit"><i class="fa fa-send"></i>Send</button>
                    </form>

Use the following Bootstrap grid structure for all importers:


                    <div class="col-md-5 col-lg-6">
                      <div class="row">
                        <div class="col-md-12">
                          <!-- Form goes here -->
                        </div>
                      </div>
                    </div>
                    <div class="col-md-1"><!--Nothing here. Just space between form and callout --></div>
                    <div class="col-md-6 col-lg-5">
                      <div class="row">
                        <div class="zd-callout zd-callout-default">
                          <!--File uploading callout -->
                        </div>
                      </div>
                    </div>

Keep in mind that all importers and file uploaders should have an assistance callout with a button to download templates, if applicable.
