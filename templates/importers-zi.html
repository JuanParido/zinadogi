Rules to complete the file to upload for a successful process:

Mandatory fields:

  • Group
  • Vendor Outline Agreement (VOA)
  • VOA Item number
  • WBS
  • In AW1, you can include Yes or leave it empty. This column serves to indicate if the Scratch PO is a Sellable Additional Work.
  • If you want to have your SPO linked in IPM, you must provide a 4th level WBS in WBS column.
  • Acc. Indicator is linked with the type of the process you're working with. It can be left empty in case it's a pure Non B2B service. Otherwise, you can use the following categories (specially when working with Change Requests SPOs)
  • 9 for Risk Contingency
  • 93 for NCC Resources
  • 94 for NCC Delivery
  • 95 for NCC Execution
