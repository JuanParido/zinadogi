How to use callouts

Use the following structure to implement a callout on your view: 

                          <div class="col-md-4"> <!--Include any desired call width-->
                            <div class="zd-callout zd-callout-default"><!--You can use default, info, warning, danger or success-->
                              <h4>Default</h4><!--Callout title-->
                                <p>Lorem ipsum dolor sit amet, consectetur adipisicing elit. Ipsa, ab consequatur! Voluptas earum provident odit error, molestiae, blanditiis accusamus rerum, ducimus consectetur repellat voluptate dicta suscipit laborum delectus accusantium nisi.</p>
                              </div><!--Callout text-->
                          </div>

How to use alerts

Use the following structure to implement alerts in your view: 

                          <div class="col-md-4"> <!--Include any desired call width-->
                            <div class="alert alert-default alert-dismissible" role="alert"><!--You can use default, info, warning, danger or success. Always include the role attribute.-->
                            <!--Include alert-dismissible class whether you like to use dismissable alerts. Otherwise don't include it.-->
                              <i class="fa fa-bolt"></i>Default<!--Alert text. You can also include FontAwesome icons at the beginning-->
                                <button class="close" type="button" data-dismiss="alert" aria-hidden="true"><i class="fa fa-times"></i></button><!--Closing button. Include only when alert-dismissible class has been included as well.-->
                                <!--Always include data-dismiss and aria-hidden attributes-->
                          </div>

How to use modals

Use the following structure to implement modals in your view: 

                          <button type="button" class="btn btn-default" data-toggle="modal" data-target="#myModal">Launch me!</button>
                          <!--Use data-toggle and data-target in anchors or buttons to trigger the modal.-->
                          <div id="myModal" class="modal fade" role="dialog">
                            <div class="modal-dialog">
                              <!-- Modal content-->
                              <div class="modal-content default"><!--You can use default, info, success, warning and danger classes here.-->
                                <div class="modal-header">
                                  <button type="button" class="close modal-close" data-dismiss="modal"><i class="fa fa-times"></i></button>
                                  <h2 class="modal-title tile-title"></h2>
                                </div>
